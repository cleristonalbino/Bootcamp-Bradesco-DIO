# Mentor de Carreira em Tecnologia - Powered by NotebookLM

## Descrição
Este projeto foi desenvolvido durante o **Bootcamp de GenAI e Dados do Bradesco (DIO)**. O objetivo foi criar um assistente virtual personalizado utilizando o **Google NotebookLM** para atuar como um mentor para iniciantes na área de tecnologia e software.

## Tecnologias Utilizadas
* **Google NotebookLM**: Para a criação do modelo de IA fundamentado em fontes específicas.
* **Curadoria de Conteúdo**: Seleção de livros clássicos de computação e vídeos educativos do YouTube.

## Processo de Criação
1.  **Coleta de Fontes**: Reuni materiais de referência como "Ciência da Computação uma visão abrangente", "Fundamentos da Programação, Ascensio", "O Programador Pragmático", "Redes de Computadores, Andrew Tanenbaum" e outras fontes de relevância assim como com canais relevantes de tecnologia, "Programe seu Futuro".
2.  **Ingestão de Dados**: Realizei o upload dos PDFs e inserção dos links no NotebookLM.
3.  **Configuração do Prompt**: Instruí a IA para atuar como um "Professor Mentor", focando em explicações didáticas e estruturadas para quem está começando (nível júnior/estágio).

## Resultados
O assistente é capaz de:
* Explicar conceitos complexos de algoritmos de forma simples.
* Sugerir trilhas de estudo baseadas nas fontes confiáveis fornecidas.
* Tirar dúvidas sobre carreira em Machine Learning e Engenharia de Software.
